# /what-next — предложить следующие шаги наибольшей ценности

Иногда агент останавливается и ничего не предлагает. Этот навык — ответ одной командой на простой вопрос
владельца — **«что дальше?»** — план следующих шагов, ранжированный по ценности для видения владельца.

## Процедура

### Шаг 1. Сориентируйся (дёшево, без глубокого погружения)
Прочитай `GOAL.md` (видение), `MASTER_PLAN.md` (текущая фаза), `STATUS.md` (где мы), пройдись по
открытому беклогу: `bugs/`, `ideas/`, `plans/`, `homeworks/` без тега `DONE`, плюс неотвеченные
`interviews/`.

### Шаг 2. Ранжируй по ценности
Упорядочь шаги-кандидаты по **ценности для видения** согласно `PHILOSOPHY.md`: Парето (немногое
жизненно важное), матрица Эйзенхауэра (важно × срочно), эффекты второго порядка (что разблокирует больше
всего будущей работы). Отметь примерную трудоёмкость каждого.

### Шаг 3. Ответь в чате
1. **ОДИН следующий шаг** — наибольшая ценность, и *почему именно он* (свяжи с GOAL/MASTER_PLAN).
2. **2–4 запасных** — по строке на каждый, с ценностью/трудоёмкостью.
3. **Заблокировано на владельце** — открытые интервью/домашки, если есть.

### Шаг 4. Предложи начать
Предложи немедленно взять верхний шаг; по подтверждению владельца (или в автономном цикле) — начинай.

## Правила
- Никогда не отвечай «делать нечего»: пустой беклог означает — предложи `/check-backlog` или
  `/refresh-context`, работу над качеством/долгом или `/interview` для пополнения видения.
- Будь конкретен: шаги с именами и файлами, а не общие слова.
