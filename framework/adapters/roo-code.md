# Adapter: Roo Code (archived — superseded by Zoo Code)

> Roo Code was archived by its team in April 2026; its community successor is **Zoo Code**
> (fork chain: Cline → Roo Code → Zoo Code), which keeps Roo Code's paths and settings verbatim.
> **Use `zoo-code.md`** — it is byte-compatible with Roo Code installs (`.roo/rules/`,
> `.roo/commands/<name>.md`, native `AGENTS.md`, `roo-cline.*` settings).

For a project still running the archived Roo Code extension, apply the Zoo Code adapter unchanged and
record `.kaif/kaif.json` → `agent: "roo-code"`.
