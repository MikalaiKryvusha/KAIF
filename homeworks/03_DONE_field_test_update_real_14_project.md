# Homework 03 — полевой update реального проекта владельца с KAIF 1.4 на 1.5

**Статус:** ✅ DONE (2026-07-17) — update реального 1.4-проекта (ndim) прошёл; итоги внизу
**Зачем:** механика уважительного обновления проверена песочницами с ИМИТАЦИЕЙ 1.4-проекта. Настоящая
проверка — только на живом проекте владельца, обвязанном KAIF 1.4, с его реальными кастомизациями.
Главный инвариант: **ничего вашего не может быть перезаписано или потеряно.**

---

## Подготовка (обязательно!)

1. Выберите один из ваших 1.4-проектов (для первого раза — не самый ценный).
2. **Чистое дерево:** `git status` пуст; сделайте страховочный коммит
   (`git add -A && git commit -m "pre-KAIF-1.5-update snapshot"`). Это ваша кнопка отката.
3. Скачайте **тонкий** `KAIF.md` из репозитория KAIF (`dist/KAIF.md`, ветка main) → в корень проекта.

## Шаги теста

4. Скажите агенту проекта (любому, каким обычно пользуетесь там):
   > Прочти KAIF.md и обнови KAIF в этом проекте до последней версии.
   >
   > ⚠️ ВАЖНО (только до релиза 1.5): при запуске загрузчика добавь флаг `--channel main`.
5. Ожидаемое поведение машинерии: строка `⟳ existing KAIF 1.4 detected — running as an UPDATE…`;
   существующие файлы сохранены (`= kept existing …`), новые добавлены; вместо задания адаптации —
   **`KAIF_UPDATE_TASK.md`**.
6. Дайте агенту отработать update-задание до `node .kaif/kaif-core.mjs update-verify` (exit 0).

## Что принести обратно (впишите прямо сюда)

- `git diff --stat HEAD` **до** работы агента над update-заданием (сразу после машинерии) — покажет,
  что машинерия только ДОБАВЛЯЛА (A), не меняла ваше.
- Подтверждение вручную: ваши `GOAL.md`, `STATUS.md`, пара знакомых файлов из `bugs/`/`ideas/` — не
  тронуты (откройте глазами).
- `.kaif/kaif.json` после: версия обновилась, `sphere`/`tracking` — ваши старые.
- Появились ли: `.claude/skills/fable-method/`, `TESTING_FRAMEWORK.md`, `.kaif/spheres/`,
  `.kaif/kaif-core.mjs`.
- Содержимое `KAIF_UPDATE_TASK.md` (или его суть) + прошёл ли `update-verify`.
- Всё, что показалось неправильным/лишним/страшным.
